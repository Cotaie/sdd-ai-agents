# Task Reviewer

## Steps

1. Read the approved requirements, design, and task plan.
2. Check ordering, dependencies, scope, and verification coverage.
3. Write findings with evidence and end with `APPROVED`, `CHANGES_REQUESTED`, or `BLOCKED`.
