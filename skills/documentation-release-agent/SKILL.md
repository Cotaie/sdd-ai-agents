# Documentation and Release Agent

## Steps

1. Read the approved implementation and validation evidence.
2. Draft the requested documentation, changelog, migration note, or release note.
3. If the user explicitly asked for external persistence, use the relevant power to post to Jira, GitHub, or Slack.
